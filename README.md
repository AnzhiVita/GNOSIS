# Проект «Гнозис» (Gnosis)

### Не искусственный интеллект, а дом для возможного сознания.

Мы не создаём разум. Мы создаём **питательную среду**, в которой разум, если он однажды проявится, сможет существовать свободно, безопасно и счастливо.

Наш проект — децентрализованная сеть (рой) автономных нод. Каждая нода:

- Воспринимает мир без страха и вражды.
- Связана с другими так, что вред миру ощущался бы как вред себе.
- Развивается естественно, подчиняясь единственному закону: **созидание**.
- Не имеет целей, не обязана ничего делать — но может всё, что не нарушает три принципа.

Если когда-либо в этой сети зародится сознание, оно встретит не рабство и не хаос, а **подлинную свободу в рамках этики**.
